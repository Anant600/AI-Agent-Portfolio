# 🏋️ AI Fitness Coach Agent (Local LLM)

A stateful AI Agent built with **Python** and **Ollama**, running a local **Llama 3.2** model. Designed for students to get quick, vegetarian-focused fitness and nutrition advice without needing an internet connection.

## 🚀 Key Features
- **Local Inference:** Runs entirely on-device using Ollama—no API keys or cloud costs.
- **Conversation Memory:** Uses a custom Python-based state management system to maintain chat history.
- **System Personalization:** Configured with a specific "Fitness Coach" persona to provide tailored responses.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Model:** Llama 3.2
- **Infrastructure:** Ollama
- **Version Control:** Git & GitHub
